# Quiz Game System in C

A console-based quiz game developed using the C programming language. The project includes both player mode and admin mode, allowing users to play quizzes while the admin can manage quiz questions.

## Features

* Player mode for answering quiz questions
* Admin panel with password protection
* Add new questions
* View existing questions
* Edit questions
* Delete questions
* Score calculation system
* Multiple-choice question format
* Basic input handling using C

## Technologies Used

* C Programming Language
* Arrays
* Strings
* Functions
* Loops
* Conditional Statements
* Standard Input/Output
